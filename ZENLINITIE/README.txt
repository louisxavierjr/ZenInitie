II - Projet en bref
=============
Le projet était axé sur le développement numérique du jeu « Zen l’initié », un jeu en bois sorti en 1997.
Il est disponible sous deux version  : en mode console et en mode graphique

III - Installation
===========
Pour générer l'executive jar de l'application, il faut exécuter la commande suivante dans le repertoire du jeu ZenLinitie : ant jar
/!\ ATTENTION /!\ : Ne jamais séparer le dossier jar du dossier data si vous souhaitez jouer en mode graphique, toutes les images risquent de ne pas se charger.

III - Lancement du jeu Zen l'initié
=========================
A / Ouvrir un terminal et se placer dans le dossier jar, disponible dans le même dossier où se trouve ce ficher README.txt

B / Ecrire dans le terminal la commande correspondante à votre choix : 
B1 -> Mode console : "java -jar ZenLinitie.jar CONSOLE"
B2 -> Mode graphique : "java -jar ZenLinitie.jar GRAPHIQUE"
N.B : Il n'y a que ces deux commandes pour exécuter le jeu

C / Il ne vous reste plus qu'à tester !
